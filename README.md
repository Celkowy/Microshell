# Microshell

Shell program in ANSI C language with Makefile written.
```
Write:
start - to compile
./microshell - to start a program
make clean - to clear 
```

```
Available commands:
help - shows the list of available commands and features
cd - change directory
mkdir - make directory
rmdir - remove empty directories
clear - clear the terminal screen
exit - cause normal process termination
feedback when command is unknown ect.
```
