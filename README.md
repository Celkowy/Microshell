# Microshell

Shell program in ANSI C language with Makefile.

Write:
```
make - to compile
./microshell - to start a program
make clean - to clear
```

Available commands:
```
help - show the list of available commands and features
cd - change directory
mkdir - make directory
rmdir - remove empty directories
clear - clear the terminal screen
exit - cause normal process termination
```

Feedback when:
```
- command is unknown
- deleting not existing directory or the directory is not empty
- creating a directory with name already existing
- changing directory to not existing one
```
